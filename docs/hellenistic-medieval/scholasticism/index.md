# 经院哲学总览

「经院哲学」（拉丁文 *philosophia scholastica*；英文 scholastic philosophy）不是一组人人共享的教义，而是在中世纪学校和大学中形成的教学、释义与论辩传统。它大致以十一世纪学校复兴为起点，在十三世纪大学制度和亚里士多德著作全面传入后达到高峰，至十四世纪出现更明显的学派分化。这里把波爱修、伪狄奥尼修和九世纪的爱留根纳作为前史，主体从安瑟尔谟、阿伯拉尔写到托马斯·阿奎那、波那文都拉、邓斯·司各脱、奥卡姆的威廉和迈斯特·艾克哈特。

「经院」首先指一种制度位置和工作方法，不等于思想僵化。教师围绕权威文本作讲解（*lectio*），从文本冲突中提出问题（*quaestio*），列举相反理由并作答辩（*disputatio*）。阿奎那《神学大全》中「异议—相反权威—正文回答—逐条答复」的结构，是这种训练的成熟形式。权威并不使论证停止：经文、教父、亚里士多德和法学文本之间经常表面冲突，教师必须区分词义、命题范围和论证层次，说明它们能否相容。

## 六个贯穿问题

第一，信仰内容能否成为严格理解的对象？安瑟尔谟从已信的内容出发寻求内在理由，阿奎那区分哲学论证可知的「信仰前导」与只能由启示得知的奥迹，奥卡姆则进一步缩小自然神学中可作必然证明的范围。「信仰与理性」不是一次性的服从关系，而包含起点、方法、对象和确定性程度四项区别。

第二，普遍概念依据什么而有效？若只有个别的人和马存在，「人」和「马」怎样共同指称多个对象并进入科学命题？若共相是独立实在，又怎样避免把柏拉图式分离理念重新引入基督教创造论？从阿伯拉尔的状态与谓述，到阿奎那的事物中形式和理智抽象，再到司各脱的共同本性和奥卡姆的心灵符号，答案同时涉及存在论、语义学和认识论。

第三，受造物的本质与其「是」是什么关系？阿维森纳对本质与存在的分析深刻影响拉丁哲学。阿奎那认为有限实体的本质不包含其存在，受造物因参与存在而实有；只有上帝是自存的存在本身。司各脱和奥卡姆不接受阿奎那式解释的全部部分，「存在」遂成为真题中不能只译为「有」的体系概念。

第四，人的知识从感觉、抽象还是照明而来？阿奎那以感官和能动理智说明普通认识，波那文都拉保留对确定真理的神圣照明，司各脱和奥卡姆区分直观认识与抽象认识。这些理论争论的不是人是否偶尔有神秘经验，而是心灵怎样取得个体、普遍与必然知识。

第五，上帝存在和属性可以证明到什么程度？安瑟尔谟从「大于它者不可设想」推进，阿奎那从经验到的运动、因果和秩序作五路论证，司各脱试图证明第一有效因与无限存在者，奥卡姆则质疑若干传统推论的必然性。把它们统称为「证明上帝」会漏掉证明起点、因果原则和结论强度的差别。

第六，理智还是意志在行动中优先？阿奎那认为意志以理智所呈现的善为对象，同时承认意志有自身活动；司各脱更强调意志在可选择对象之间自我决定的能力；奥卡姆则以偶然性和神圣全能重画道德秩序。这个问题连接自由、自然法、功德和近代道德主体，但不能简单写成「理性主义被意志主义取代」。

## 范围说明

本单元聚焦拉丁西方经院哲学，不把伊斯兰哲学和犹太哲学仅当作附注，却也无法替代它们各自传统的完整历史。法拉比、阿维森纳、阿威罗伊、迈蒙尼德等人在此主要按其对拉丁讨论的具体作用出现。中世纪晚期还包括布里丹、奥特雷库尔的尼古拉、威克里夫等重要作者；为配合现有考试范围，本轮以司各脱、奥卡姆和艾克哈特结束。
