# 教父哲学总览

「教父哲学」不是一个像斯多亚派那样具有统一学说和组织的学派，而是早期基督教作者在护教、释经、教义争论、讲道和灵修中使用哲学论证的思想传统。其大致范围可从二世纪护教士写到五世纪的奥古斯丁；若按希腊教父传统，则还会延伸到七至八世纪。这里以中山大学西方哲学史的考查范围为准，重点说明希腊哲学概念怎样进入基督教问题，并集中讨论奥古斯丁。

## 五组核心问题

第一，信仰与理性怎样相关？早期作者既要反对把福音还原为某一哲学学派，又必须回应异教批评、解释经文并澄清教义。「雅典与耶路撒冷」和「哲学是真理的预备」不是简单的反理性与亲理性之争，而是关于哲学权限和用途的不同判断。

第二，一个超越世界的上帝怎样创造并维持世界？教父传统借用柏拉图主义的非物质实在语言，却把形式置于上帝的理智中，并以自由的从无创造区别于柏拉图《蒂迈欧篇》的工匠整序和新柏拉图主义的本原流溢。

第三，人怎样认识真理？奥古斯丁反驳学园怀疑主义，以逻辑真理、显现和自我意识寻找确定性；神圣照明（illuminatio divina）又要说明可变心灵为什么能够判断不变真理。照明是否提供认识内容，学界并无单一解释。

第四，如果一切存在都来自善的上帝，恶从哪里来？奥古斯丁用「善的缺失」（privatio boni）否认恶是与善并立的实体原则，以意志背离更高善解释道德恶；他晚年对原罪和恩典的强调又限制了堕落后意志自行归正的能力。

第五，基督教怎样理解政治和历史？《上帝之城》不以罗马帝国的兴亡衡量神意，也不把任何现存国家等同上帝之城。两座城由两种根本之爱构成，在现世历史中交错，且都可能使用暂时的政治和平。

## 复习范围说明

本单元不是早期基督教神学史的完整替代。三位一体和基督论争论只介绍理解哲学术语转换所需的范围；不展开各地区教会史、礼仪史和全部异端谱系。考研短题作答应从问题、概念、论证和限定四步展开，尤其要避免把奥古斯丁的结论缩成「信仰高于理性」「恶不存在」或「教会高于国家」。
