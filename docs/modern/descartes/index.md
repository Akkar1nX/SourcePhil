# 笛卡尔总览

勒内·笛卡尔（René Descartes, 1596—1650）试图为自然科学建立确定基础。他没有凭空发明「主体」或完全抛弃经院哲学：方法怀疑吸收古代怀疑主义的压力，「我思」有奥古斯丁等前史，实体、实在性与上帝证明使用经院概念；但他重新安排这些资源，使第一人称思维的确定性先于身体和外部世界知识，并以广延、形状和运动重构自然。

把笛卡尔只概括为「唯理论者」会遗漏两点。第一，他的数学物理、光学、生理学和气象学不是形而上学的附录，而是整个方法计划所要支持的知识。第二，他晚期关于身心联合、情感和德性的讨论表明，人不是日常生活中寄居机器的纯心灵。形而上学区分与具体人的联合必须同时说明。

## 著作地图

| 著作 | 时间 | 主要任务 |
| --- | --- | --- |
| 《指导心灵的规则》 | 约 1620 年代，未完成，身后出版 | 直观、演绎、次序和普遍数学方法 |
| 《世界》／《论人》 | 约 1633 年完成，身后出版 | 机械宇宙论、人体和感觉模型 |
| 《谈谈方法》及三篇论文 | 1637 | 以法文说明方法、临时道德和科学成果；附《屈光学》《气象学》《几何学》 |
| 《第一哲学沉思集》 | 1641；1642 第二版 | 依「沉思次序」建立自我、上帝、物体和身心区分；附反驳与答辩 |
| 《哲学原理》 | 1644 | 以教科书形式组织形而上学和自然哲学 |
| 《论灵魂的激情》 | 1649 | 身心联合、情感机制、意志训练和慷慨 |

《谈谈方法》不是《沉思》的通俗缩写，《哲学原理》也不是简单重复。三者面对不同读者并采用不同论证次序：自传式方法说明、第一人称沉思练习和原则式教科书。引用「我思」或上帝证明时应注明文本。

## 体系中的七个枢纽

第一，方法怀疑是主动而暂时的检验，不是主张一切知识不可能。笛卡尔把可疑意见当作假的，以寻找绝对不可动摇者；日常行动则由临时道德维持。

第二，「我思」的确定性发生在思维活动当下。怀疑者要怀疑便必须思维，思维发生时这个思维者不能不存在。但从此能否立即得到一个持续、自同一、无身体的实体，仍需后续论证。

第三，清楚（*clarus*）与分明（*distinctus*）是确定知识的标准。它们不是主观强烈确信：清楚指观念显现于专注心灵，分明指其内容与其他内容严格分开。《哲学原理》I 45 对二者有专门界定。

第四，有限心灵中的无限上帝观念承担保证功能。笛卡尔论证，一个无限完满者观念不能以有限自我作为充分原因；上帝存在且不欺骗，才使清楚分明知觉在不再当下注意时仍可作为知识。该论证及其是否循环是体系焦点。

第五，物体的本质是广延（*extensio*），心灵的本质是思维（*cogitatio*）。两者可被清楚分明地分别理解，因而是实在有别的受造实体。物理解释相应诉诸大小、形状、位置与运动，不诉诸经院实体形式和实在性质。

第六，具体的人是心身实质联合。饥饿、疼痛、情感不是纯理智对身体状态的外部观察，而以混合方式呈现联合的益害。笛卡尔既坚持区分，也拒绝把身体只当船而心灵只是船长。

第七，错误来自意志超出理智。有限理智只清楚呈现部分内容，意志却能对未明事项判断；停止判断即可避免错误。这使认识论与自由、责任和德性相接。

## 在近代哲学史中的位置

斯宾诺莎接受几何式秩序和清楚观念，却拒绝两个受造实体，主张只有一个实体；莱布尼茨反对物体只由广延构成，并以单子和预定和谐处理活动与身心关系。洛克批评天赋观念，仍继承观念作为直接认识对象的语言；贝克莱和休谟会进一步质疑物质实体与自我实体。康德则认为笛卡尔从「我思」推出灵魂实体犯了先验心理学的错误，同时把自我意识转化为经验统一的条件。

因此，笛卡尔不是「唯理论」名单上的第一项，而是近代关于观念、确定性、实体、因果和心身问题的共同起点之一。后继者往往通过改写其问题而建立自己的体系。
