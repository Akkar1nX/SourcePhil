# 霍布斯总览

托马斯·霍布斯（Thomas Hobbes, 1588—1679）把自然、感觉、欲望、语言和政治放进运动与因果的统一说明中。他反对非物质实体和经院「无形形式」，却没有把政治直接从物理定律机械演绎出来：政治科学还依赖定义、行动者的恐惧与欲望、契约语言和代表制度。

英国内战是《利维坦》（*Leviathan*, 1651）的直接背景，但霍布斯的政治问题早已形成。《法律要义》约 1640 年流传，《论公民》1642 年出版；《论物体》《论人》后来构成其「哲学原本」计划。内战强化了秩序问题，不能把全部理论简化为受战乱惊吓后的专制辩护。

## 著作与问题

| 著作 | 时间 | 主要问题 |
| --- | --- | --- |
| 《法律要义》 | 1640，手稿流传 | 人性、自然法和政治体 |
| 《论公民》 | 1642 | 自由、统治与宗教 |
| 《利维坦》 | 1651 | 人、契约、主权、基督教国家与黑暗王国 |
| 《论物体》 | 1655 | 方法、几何、物体和运动 |
| 《论人》 | 1658 | 感觉、光学与人类能力 |

《利维坦》四部分依次讨论人、国家、基督教国家和「黑暗王国」。只读第 13—18 章会掌握契约主干，却看不到语言、人格代表、宗教权威和错误哲学为何属于同一政治工程。

## 体系坐标

霍布斯比笛卡尔更彻底地把机械论用于人：感觉与思维是身体运动，不需非广延心灵。与斯宾诺莎相比，两者都否认不受因果制约的自由意志，但霍布斯以外在阻碍界定自由，斯宾诺莎则把自由放在依自身本性行动和理解必然性中。洛克继承自然权利、同意和契约问题，却拒绝主权不可分割的结论；卢梭会批评霍布斯把社会化欲望投射到自然人。
