# 近代哲学总览

本目录以十六至十八世纪欧洲哲学为范围，起点不是一个精确年份。文艺复兴人文主义、宗教改革、古代怀疑主义复兴、殖民扩张和新自然科学改变了文本、权威与经验的条件；笛卡尔、霍布斯、斯宾诺莎、莱布尼茨等人再以不同的实体和因果理论回应机械自然；洛克、贝克莱、休谟等人围绕观念来源、外部世界、自我和因果收紧知识限度；启蒙思想与常识哲学又把这些争论带入道德、政治、宗教和公共判断。康德的批判哲学以这组问题为直接前史。

「唯理论」与「经验论」是有用但较晚形成的分类。笛卡尔、斯宾诺莎和莱布尼茨并非不使用经验，洛克、贝克莱和休谟也并非拒绝理性论证；分类主要提示他们对知识来源、必然性和天赋结构的不同判断。后续比较应围绕共同问题展开，而不是把两份人物名单当作相反阵营。

## 后续编写次序

下一阶段为启蒙主义及康德以前的直接问题背景，包括法国启蒙运动、自然法与道德理性主义、卢梭、孟德斯鸠等。现有[核心问题比较](core-comparisons.md)已把实体、观念、因果、心身、自我、自由和政治贯通，后续将在此基础上加入公共理性、历史、文明和进步问题。

## 本阶段应保留的比较坐标

| 问题 | 文艺复兴与转型 | 笛卡尔的重新规定 |
| --- | --- | --- |
| 权威 | 语文学核对古典与经文；宗派争论解释权 | 以方法主体重检意见，清楚分明知觉成为判断标准 |
| 怀疑 | 蒙田用判断悬置限制独断 | 把夸张怀疑作为通向第一确定性的临时方法 |
| 自然 | 亚里士多德主义、自然魔法、经验史和数学化并存 | 以广延、形状与运动建立机械自然 |
| 人 | 可塑性、罪与恩典、政治自由各有语境 | 思维实体与广延身体实在有别，具体人又是二者联合 |
| 方法 | 培根从自然史、排除和实验逐步上升 | 从简单明证出发，依分析与演绎次序推进 |
| 政治与伦理 | 马基雅维利研究建国、冲突和公共自由 | 以判断、意志、激情治理与慷慨构成个人伦理，未完成系统政治哲学 |
