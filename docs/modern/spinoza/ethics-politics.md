# 情感、自由、政治与宗教

## 努力与欲望

每一事物都努力（*conatus*）在自身存在中持续，且这种努力就是其现实本质（E3p6—p7）。在人同时涉及心灵与身体时称欲求（*appetitus*），有欲求意识时称欲望（*cupiditas*）。人不是先判断某物善才欲求，而是因欲求它才称其为善；但理性能够重组何者真正增强行动能力。

喜悦是向较大完善或行动能力的过渡，悲哀是向较小完善的过渡；爱是伴随外在原因观念的喜悦，恨是伴随外因观念的悲哀。情感定义把评价、身体变化和因果想象结合，不是只给感觉命名。

## 主动与被动

当我们是一个结果的适当原因时行动，只有不适当原因时受作用。被动情感来自局部相遇和混乱观念，主动情感来自适当认识。一个真观念不会因真便自动压倒强烈情感；只有作为情感的更强相反情感才能抑制情感（E4p7）。

理性通过理解共同原因、把情感从单一外物想象转入更广因果网络，减少被动性。目标不是斯多亚式绝对无情感；E5 序言也批评心灵对身体拥有绝对统治。自由人仍有喜悦、欲望和爱，只是更少由外因摆布。

## 自由、德性与知性之爱

斯宾诺莎否认意志是独立能力；每个肯定和选择都有原因。人自以为自由，是因为意识到欲望却不知道决定欲望的原因。自由不是打断因果，而是更充分由自身本性的适当原因行动。

德性（*virtus*）等同人的能力（*potentia*），按理性保存和发展自身。理性自利并非狭隘竞争：最有益于人的是生活在理性中的他人，合作和友谊增强共同能力。自由人少想死亡，其智慧是对生命的沉思（E4p67）。

对上帝的知性之爱（*amor intellectualis Dei*）由第三种认识产生；心灵理解自己和个别物是神圣自然的必然表达，并分享上帝对自身的无限知性爱。幸福不是德性的奖赏，而就是德性和主动认识本身（E5p42）。

## 自然权利与国家

斯宾诺莎同霍布斯一样把自然权利（*jus naturale*）联系到实际能力：每一事物有多大力量，便有多大自然权利。自然状态中理性规则没有自动强制力；个体为更安全有力的生活联合，形成共同权力。

但斯宾诺莎不把权利视为可由一句承诺完全让渡。人的判断和欲望不能任由命令控制，国家权力以实际服从能力为边界；以恐惧维持的国家脆弱。国家目的不是把人从理性存在变成牲畜，而是安全、和平及使心灵和身体自由发展。

《神学政治论》倾向民主，因民主最接近众人共同保有自然权利；未完成的《政治论》更注重君主、贵族和民主制度怎样利用真实情感而稳定。斯宾诺莎不是假定公民都理性，而是要求制度在普通欲望条件下运行。

## 经文、预言与思想自由

斯宾诺莎以历史—语文学方法解释经文：先研究语言、作者、编辑、受众和文本传承，再确定教义。先知的长处在生动想象和道德力量，不在超常哲学知识；经文核心是服从、正义与仁爱，而非自然学和形而上学。

哲学追求真理，神学促进虔敬服从，二者目标不同。国家若压制思想，不能真正控制内心，反会制造伪善和冲突；只要言论不直接破坏法律行动，应保留「哲学思考的自由」（*libertas philosophandi*）。这种宽容仍服从公共和平，也未发展为无限制表达权。
