# 斯宾诺莎总览

巴鲁赫／本笃·斯宾诺莎（Baruch／Benedictus de Spinoza, 1632—1677）把笛卡尔的实体、属性和几何秩序推向一元论：只有一个无限实体，即上帝或自然；心灵与身体不是两个实体，而是同一事物在思维和广延属性下的表达。伦理自由因而不能是脱离因果的任意选择，而是从自身本性行动并理解使我们成为现在这样的原因。

斯宾诺莎生于阿姆斯特丹葡萄牙犹太社群，1656 年受到逐出社群的 *herem*。现存文书列举「可憎异端」等控告，却未记录完整具体学说，不能把后来《伦理学》的每项命题倒填为逐出原因。他靠研磨透镜等工作生活，与荷兰共和国的学者和政治网络通信。

## 著作地图

| 著作 | 时间 | 主要任务 |
| --- | --- | --- |
| 《知性改进论》 | 早期，未完成 | 方法、真观念和最高善 |
| 《笛卡尔哲学原理》 | 1663 | 以几何方式讲解笛卡尔；唯一以本名出版的著作 |
| 《神学政治论》 | 1670，匿名 | 经文解释、预言、宗教、国家和思想自由 |
| 《伦理学》 | 1677，身后出版 | 实体、心灵、情感、受制与自由 |
| 《政治论》 | 1677，未完成 | 国家形式、制度稳定和民主 |

《伦理学》五部分从上帝、心灵、情感起源，推进到人的受制与理智力量。定义、公理、命题、证明、绎理和附释的几何形式不表示每个定义无争议；附释和序言常揭示证明背后的宗教、心理和论战目标。

## 哲学史位置

斯宾诺莎继承笛卡尔的清楚观念和机械自然，却拒绝受造实体及心身互动；继承霍布斯对欲望、因果必然和自然权利的重视，却不把国家目的缩为安全，也更积极捍卫思想自由。莱布尼茨承认其体系力量但反对必然论与一实体说。德国观念论和浪漫主义重新评价斯宾诺莎，使「斯宾诺莎主义」从无神论指控转为体系哲学的重要资源。
