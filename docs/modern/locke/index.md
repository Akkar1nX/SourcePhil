# 洛克总览

约翰·洛克（John Locke, 1632—1704）以观念来源和认识限度为哲学起点，又以自然权利、财产、同意和宽容重构政治义务。《人类理智论》（*An Essay concerning Human Understanding*, 1690）不是说一切知识只是被动感觉：心灵从感觉和反省获得简单观念，再主动比较、组合、抽象，并以直觉和证明认识观念间关系。

洛克在牛津受教育，参与医学与实验科学网络，长期服务沙夫茨伯里伯爵，因政治风险流亡荷兰，光荣革命后返英。《政府论两篇》主要写作时间和出版策略仍有研究争议，不能只说它是 1688 年革命后的即时辩护。

## 哲学史位置

洛克反对笛卡尔式天赋原则，却保留观念作为直接认识材料和物质微粒假说；贝克莱会利用其观念论语言攻击物质基质，休谟再把观念来源原则推向因果、自我和宗教批判。莱布尼茨逐章回应洛克，认为经验不足以说明必然真理。政治上，洛克改写霍布斯的自然状态与契约，同时影响美国革命、自由主义和财产权论述；其殖民行政关系也使普遍权利与帝国实践的张力不能略去。
