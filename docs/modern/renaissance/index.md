# 文艺复兴与近代转型总览

「文艺复兴」（Renaissance，意大利语 *Rinascimento*，再生／复兴）通常指十四至十六世纪欧洲文化、知识与艺术的一系列变化，而不是某一天开始的哲学时代。十九世纪的雅各布·布克哈特把意大利文艺复兴描写为个人觉醒和现代国家诞生；后来的研究则强调，它既复兴古典文献，也延续大学经院哲学、基督教制度和中世纪城市传统。因而，「中世纪结束以后人突然发现人」不是可靠的分期说明。[^1]

哲学史中的关键变化首先发生在知识材料和工作方式上。希腊文学习扩展，柏拉图、普罗提诺、塞克斯都·恩披里柯等文本得到翻译和印刷；人文主义者以语文学核对文本、历史语境和词义，批评只凭拉丁译本进行的概念推演。亚里士多德主义并未随之消失，帕多瓦等大学的自然哲学和逻辑研究反而十分活跃。柏拉图主义、亚里士多德主义、斯多亚主义、伊壁鸠鲁主义和怀疑主义由此在新的宗教与政治条件中重新相遇。

## 六条主线

第一，古典复兴不是复制古代。人文主义（*studia humanitatis*，人文学科）以语法、修辞、诗歌、历史和道德哲学训练公共表达与实践判断；古代材料被放入城市共和国、宫廷、教会改革和教育竞争中，因而获得新的用途。

第二，语言成为哲学问题。洛伦佐·瓦拉借拉丁文体、历史语义和文献证据揭示《君士坦丁赠礼》的时代错误，也批评经院拉丁文和亚里士多德范畴的某些用法。语文学并非只给哲学准备材料，它会追问一个概念在何时、由谁、为了什么而使用。

第三，人的地位被重新表达，但并未脱离神学。皮科《论人的尊严》把人的未定位置和自我塑造能力置于上帝创造的宇宙中；宗教改革则以罪、信仰、恩典和良知重新限制人的能力。所谓「人文主义发现自主主体」只能作为待检验的解释，不能代替文本。

第四，政治思想更多分析权力自身的运行。马基雅维利讨论共和国、军队、时机、冲突和建国行动，不再把政治论述完全组织为统治者德性清单。但他仍使用罗马史、古典德性和共同体自由的语言，不能简单归为「目的证明手段」。

第五，自然研究的权威和对象同时改变。亚里士多德自然学继续存在，柏拉图主义、炼金术、占星术和自然魔法也参与探索自然力量；解剖、航海、工程、观测与数学化逐渐改变可信证据。现代科学不是从「迷信」一步跨入「理性」，而是从相互交叠的实践中形成新的筛选标准。

第六，古代怀疑主义进入近代认识论。塞克斯都著作的拉丁译本使皮浪式论证广泛传播；蒙田用它检验理性自负、宗教争论和习俗判断。笛卡尔的方法怀疑会把怀疑变成寻找不可动摇起点的工具，其目标和皮浪主义的判断悬置并不相同。

## 阅读定位

若以近代哲学主干为目标，本单元不要求把所有人文主义者记成完整体系。更重要的是理解：笛卡尔面对的不是一个已经消失的「中世纪」，而是经院课程、宗教分裂、新天文学、机械技术、怀疑主义和知识改革方案同时存在的十七世纪世界。培根提供一种经验与组织改革方案，蒙田展示怀疑和自我考察的力量，伽利略改变自然知识的数学形式；笛卡尔的第一哲学和机械自然学正是在这些问题中取得位置。
