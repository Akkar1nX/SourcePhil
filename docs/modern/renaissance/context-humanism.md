# 历史、社会与知识条件

文艺复兴最早在意大利城邦形成显著的文化样式，但欧洲各地区的时间、载体和宗教背景并不相同。佛罗伦萨、威尼斯、米兰、罗马和那不勒斯既有城市政治、商业金融和宫廷赞助，也有战争、寡头统治和教廷权力。阿尔卑斯山以北的人文主义更多与大学、宫廷学校、印刷业及教会改革结合。把一个意大利艺术史概念直接套到整个欧洲，会掩盖这种差异。

## 城市、宫廷与赞助

十四至十五世纪意大利没有统一民族国家，而由共和国、领地国家、教皇国和王国竞争。佛罗伦萨名义上保持共和国制度，却长期受美第奇家族影响；威尼斯由贵族共和国治理；米兰由维斯孔蒂和斯福尔扎家族建立领地统治。外交使节、书记官和宫廷教师需要熟练的拉丁文、历史范例与说服术，人文主义教育因而具有明确的职业和政治条件。

赞助既给学者、翻译者和艺术家提供资源，也规定题材、献辞和名望竞争。费奇诺的柏拉图翻译与美第奇网络密切相关，但所谓佛罗伦萨「柏拉图学院」是否是具有正式成员和制度的学院，研究者多持谨慎态度；它更可能是围绕费奇诺形成的赞助与交往网络。[^1]

1494 年法国查理八世入侵意大利，开启持续数十年的意大利战争。1527 年神圣罗马帝国军队洗劫罗马。战争使雇佣军、外交、共和国自由和君主统治成为迫切问题，马基雅维利的政治论述应放在佛罗伦萨共和国兴亡与外国强权竞争中理解。

## 人文主义是什么

「人文主义」（humanism）是十九世纪形成的概括词；文艺复兴时期更常说人文学科（*studia humanitatis*），包括语法、修辞、诗歌、历史和道德哲学。它首先是一套教育和文体计划，而非无神论或「以人为宇宙中心」的统一学说。许多人文主义者是神职人员或虔诚信徒，他们要求以更准确的语言阅读古典和基督教文本。

彼特拉克批评学者只会处理抽象问题却不关心灵魂和生活，推崇西塞罗、塞涅卡与奥古斯丁式自我省察。这种批评针对当时特定的大学论辩文化，并不证明他拒绝全部逻辑或亚里士多德哲学。十五世纪的「公民人文主义」解释则强调布鲁尼等佛罗伦萨人文主义者以罗马共和语言讨论积极生活和共和国自由。汉斯·巴伦把它视为米兰危机下的思想转折；后续研究指出共和修辞在此前已有来源，且人文主义也服务君主和教廷。[^2]

## 语文学怎样改变论证

人文主义口号 *ad fontes* 意为「回到源头」。这要求学习希腊文和希伯来文，比较抄本，辨别文体与历史用法。洛伦佐·瓦拉在《论伪造的君士坦丁赠礼》中依据词汇、制度称谓和历史情境，证明声称由四世纪君士坦丁授予教皇西部统治权的文书出自后世。其意义不只是反教皇：文本的真实性可以由可公开检验的语言和历史证据判断。

瓦拉还在《辩证法订正》中批评经院逻辑与形而上学脱离日常拉丁语用法，试图以修辞和语言使用重整论证。批评者会问，普通语用能否取代精密逻辑区分；支持者则认为，概念若脱离文本实践便会制造假问题。这里已经出现哲学与语文学的长期张力。

伊拉斯谟 1516 年刊行希腊文《新约》及新的拉丁译文，借文本校勘质疑《武加大译本》的若干措辞。他并非现代意义上无预设的历史批评者，其工作服务于基督徒道德革新；但经文语言一旦进入版本比较，传统权威便需要面对文本证据。

## 拜占庭学者、翻译与古典学派

十四世纪以后，曼努埃尔·赫里索洛拉斯等拜占庭学者在意大利教授希腊文。1438—1439 年费拉拉—佛罗伦萨公会议使更多希腊学者来到意大利；1453 年君士坦丁堡陷落加速部分学者与抄本西移，但不能说希腊知识只在这一年才「传回西方」。南意大利、拜占庭和中世纪翻译运动早已维持多条传播渠道。

马尔西利奥·费奇诺翻译柏拉图全集、普罗提诺及赫尔墨斯文集，以基督教柏拉图主义解释灵魂、爱与宇宙秩序。1492 年前后出版的亚里士多德希腊文印本，以及阿威罗伊、亚历山大和其他注释传统，也推动亚里士多德主义分化。所谓柏拉图「取代」亚里士多德并不符合大学课程史。

赫尔墨斯文集（*Corpus Hermeticum*）当时被费奇诺等人视为极古老的埃及智慧，因而似乎能证明柏拉图哲学与摩西启示同源。十七世纪初，伊萨克·卡索邦依据希腊语文体把其年代判定为基督教初期。这个例子显示语文学既能建立传统，也能拆解传统的古老性主张。

## 印刷与读者共同体

古腾堡约十五世纪中叶发展欧洲活字印刷。印刷降低同一文本复制成本，使版本、页码、索引和勘误能够更稳定传播；学者可围绕较一致的文本争论。它没有立刻消除手抄本，也没有自动带来思想自由。印刷需要资本、特许、市场与审查，宗教改革宣传和天主教回应都使用同一媒介。

古典作品、大学教材、地图、解剖图、宗教小册子和俗语文本形成不同读者群。知识权威由此不再只取决于口授与单本抄本，但作者、印刷商、赞助者和国家—教会审查共同塑造可见内容。

## 地理扩张与知识冲击

葡萄牙沿非洲海岸的航行、1492 年以后欧洲与美洲的持续接触，以及通往印度洋的海路，改变地图、商业和自然史材料。它们也带来征服、强迫劳动、奴役、疾病和传教。所谓「地理大发现」是欧洲叙述角度；对当地社会而言，这是入侵、殖民和跨大西洋体系形成。

新植物、动物、航线和民族志材料对古代权威提出经验压力。拉斯·卡萨斯与胡安·希内斯·德·塞普尔韦达围绕西班牙征服和原住民地位争论；弗朗西斯科·德·维多利亚及萨拉曼卡学派讨论自然法、统治权和战争正当性。这些争论仍带有传教与帝国框架，却不能从近代自然法和政治思想史中删去。

## 经济变化不等于思想的单一原因

商业、银行、远距离贸易、国家财政和价格革命改变社会关系，但「资本主义产生个人主义，个人主义产生现代哲学」证据不足。商业城市也可能保持行会、家族、教会和等级结构；人文主义在商业共和国、君主宫廷和教廷都能生长。经济条件提供职业、传播和赞助渠道，具体哲学主张仍需由文本和论争说明。
