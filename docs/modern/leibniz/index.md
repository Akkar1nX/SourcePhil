# 莱布尼茨总览

戈特弗里德·威廉·莱布尼茨（Gottfried Wilhelm Leibniz, 1646—1716）试图同时保存机械自然科学、实体内在活动、个体差异、神圣自由和世界偶然性。他接受一切物理现象应有机械解释，却认为机械结构不能解释统一、活动和知觉；真正实体必须是无部分的单子（*monas*），物体现象则建立在无数单子的有序关系上。

莱布尼茨没有出版一部包含全部体系的定本。《形而上学谈》写于 1686 年却未正式出版，《新系统》1695 年发表，《人类理智新论》因洛克去世未刊、1765 年才出版，《神义论》1710 年出版，《单子论》写于 1714 年且标题由后世编辑固定。书信、短论和不同时期术语必须相互校正。

## 四项基本原则

同一律与矛盾律规定理性真理；充足理由原则（*principium rationis sufficientis*）要求没有事物无理由如此而非彼；不可分辨者同一原则（*identitas indiscernibilium*）否认两个在全部内在性质上完全相同的实体；连续律（*lex continui*）要求自然不作无序跳跃。这些原则在逻辑、形而上学、物理和神学中用途不同，不能只列口号。

## 哲学史位置

莱布尼茨反对笛卡尔把物体本质缩为广延，认为广延可分且被动，不能构成真正统一实体；反对斯宾诺莎把一切个体化为同一实体样态，要求每个单子有内在观点；反对洛克把心灵写成无纹理白板，却同意经验是认识展开的必要条件。康德会从莱布尼茨继承分析真理、可能性和理性原则问题，同时批评其把感性差异误当作模糊理智差异。
