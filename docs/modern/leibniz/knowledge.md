# 天赋、微知觉与对洛克的批评

## 观念不能只由感觉给出

《人类理智新论》逐章回应洛克《人类理智论》。莱布尼茨接受著名经验论格言「理智中没有任何东西不是先在感觉中」，但加上「除了理智本身」（*nisi intellectus ipse*）。感觉提供场合和材料，必然性、同一、实体、原因及逻辑数学原则不能由个别感觉归纳得到。

天赋（*inné*）不是出生时显意识地背着命题。心灵像带纹理的大理石，雕像轮廓由倾向和结构预先规定，经验与注意使之显现。洛克以儿童不意识原则反驳天赋命题，若目标是现成意识内容便有力；莱布尼茨则把天赋改为能力与倾向，使争论转向这种结构如何得到独立证据。

## 微小知觉与意识连续性

微小知觉（*petites perceptions*）是未被清楚统觉的知觉。海浪轰鸣由每一微小水声组成，若完全不知觉各声，就不能知觉总声。它们解释意识状态连续过渡、习惯、情绪背景以及睡眠醒转。

莱布尼茨由此反对把心灵限于显意识。他认为没有完全无知觉的灵魂状态，也没有自然跳跃。微知觉不是现代无意识理论的完整前身，但提供了心理活动超过反思可报告内容的模型。

## 清楚、分明与观念实在性

《关于知识、真理和观念的沉思》（1684）区分：能认出对象的认识是清楚的；能列出足以区别对象的标志是分明的；标志分析到底则为适足的；同时把握全部原始成分则为直观的。人类复合概念通常无法达到完全适足直观。

一个词组可被理解，却未必有真实观念，因为其成分可能矛盾，如「最大速度」之类。真实定义要显示对象可能性，不能只给名义标志。莱布尼茨借此批评笛卡尔把清楚分明设想误当作可能性的充分保证。

想象（*imaginatio*）依连续感性图像，理智观念则能把握图像无法完整呈现的共同结构。例如没有一幅图像同时是锐角、直角和钝角三角形，却可用定义把握三角形一般本质；千边形也能由概念理解而不能清楚想象。2020 年材料应从「观念不等于影像」和「真实观念需含可能性」解释。

## 必然真理与经验

经验能告诉我们事实如此，不能单凭重复证明反面不可能。数学和逻辑的必然性来自理性原则，经验负责唤起、检验应用和提供事实材料。莱布尼茨不是否认感官知识，而是区分事实真理与必然真理的根据。

感觉观念常混乱却可清楚，例如能认出颜色但无法列出其组成标志。物理学可用测量和理论把部分感觉差异分析为更分明的运动关系，但不保证人能完全解析所有感性内容。

## 人格、自我与记忆

莱布尼茨同意意识和记忆对道德人格、奖惩非常重要，却反对洛克把人格同一完全系于可回忆意识。单一灵魂实体提供真实连续性，微知觉可跨越记忆空白；若只按实际记忆，一个人会在遗忘时不断成为不同人格。

他区分物理／真实同一与道德／人格同一：后者依反思意识和社会归责，前者由单子持续保证。这一回应预示人格同一问题不能只在心理标准和实体标准间任选，还要说明二者如何配合。
