# 单子、知觉与预定和谐

## 为什么需要单纯实体

复合物由部分构成，若真实只有复合物，统一便永远来自更小部分的聚集而无最终主体。莱布尼茨因此主张有无部分的单纯实体，即单子（*monad*）。单子不是极小物质原子：没有形状、大小和空间部分，也不能由自然分解，只能由创造开始、毁灭终止。

单子没有可供事物进入的「窗户」。《单子论》7 的意思不是单子全无关系，而是其状态变化不能由外物把性质传入；每个变化来自自身先前状态和内在原则。单子的状态相互对应，由上帝在创造时建立和谐。

## 知觉与欲求

知觉（*perception*）是在单一者中表象多样者的内部状态；欲求（*appétition*）是从一个知觉过渡到另一个知觉的内在倾向。所有单子都有知觉，却并非都有意识。统觉／自觉（*apperception*）是对知觉的反思意识，理性灵魂还能认识必然真理和自我。

单子没有空间窗口，却各从自身观点表象同一宇宙。差异来自知觉清晰度与欲求规律，没有两个单子完全相同。较低单子知觉昏暗，动物灵魂有感觉和记忆，理性精神具有反思与理性，上帝拥有完全清楚的宇宙知识。

## 磨坊论证

《单子论》17 设想一台能思维、感觉和知觉的机器按比例放大，人可走入其中。检查内部只能看到部件彼此推动，找不到能解释知觉的东西。因此知觉应在单纯实体中寻找，不能由形状和运动的机械关系解释。

论证不是说复杂机器中看不见一件叫「知觉」的零件，所以知觉神秘。它的前提是：机械解释只给出部分之间的外在关系，而知觉要求一个主体把多样统一呈现。批评者会说整体功能不必出现在任何单一零件中，走入机器只检查局部正遗漏系统组织。争议实质是统一意识能否由关系结构涌现。

## 身体与有机机器

每个生物有一个支配单子或灵魂，其有机身体由无数单子构成。自然机器与人工机器不同：机器的每个部分直到无穷仍是机器和有机结构，而人工机器的齿轮碎片不再执行同一功能。

「支配」不是灵魂物理推动身体。灵魂知觉系列与身体运动系列各按自身法则发展，表现为协调。身体也不是虚假幻觉：它是良好奠基的现象（*phenomenon bene fundatum*）或有真实单子秩序根据的聚集。学界争论莱布尼茨最终是观念论、实体质形论，还是兼有两套本体论表述。

## 预定和谐

笛卡尔身心互动难以说明非广延心灵如何推动身体；偶因论让上帝在每个场合直接配合；斯宾诺莎把心身视为同一实体样态。莱布尼茨的预定和谐（*harmonie préétablie*）主张，心灵和身体各依内在法则发展，上帝从一开始选择使二者完全对应的世界。

钟表比喻有三种协调：互相影响、修理者不断校准、制作者预先造得一致。莱布尼茨选择第三种，以保存受造实体内在活动和神圣秩序。批评者认为这把因果变成无互动的巧合；莱布尼茨则区分严格形而上学因果与现象层面可合法使用的身体因果语言。

## 单子与隐德莱希

单子也称隐德莱希（ἐντελέχεια, *entelecheia*，现实原则），因为有内在完满和活动来源。莱布尼茨借亚里士多德形式恢复机械论遗漏的统一与力，却不回到以实体形式替代所有物理机制：现象物理仍应机械解释，单子论说明这种现象秩序的形而上学根据。
