# 外部世界、自我与自由

## 物体持续存在信念

感觉只给断续知觉，不能证明对象在未感知时持续，也不能证明知觉来自独立物体。我们仍因知觉的恒常性与连贯性形成持续物体信念：离开房间再回来，陈设相似；火熄灭后余烬与先前过程连贯。

日常见解把知觉与对象同一，认为同一桌子间断出现。哲学「双重存在」理论区分短暂知觉与持续外物，让外物造成表象。休谟认为后者试图修补日常矛盾，却不能从知觉跨到绝不被经验的原因。

这不是证明无外物。自然迫使我们相信身体，怀疑论只显示理性不能给信念传统基础。因而休谟可在生活中是实在论者，在反思中拒绝形而上学证明。

## 自我是知觉束吗

反省时总遇到具体热冷、爱恨、痛乐或思想，从不遇到脱离知觉的简单自我印象。《人性论》把心灵比作各种知觉不断出现、消失和混合的舞台，但立刻提醒「舞台」比喻也不能暗示另有不变场所。

人格同一来自记忆发现知觉间因果关系、想象因相似和连续而顺畅过渡，并把关系序列误认为严格同一。休谟不是断言有一种名叫「知觉束」的新实体，而是否认经验提供简单持续灵魂观念。

《人性论》附录承认其人格同一说明有「迷宫」：他无法使关于知觉独立存在的原则与心灵真实结合原则一致。学界争论这是撤回束论，还是承认关系说明尚不充分。不能把附录一句困惑写成休谟放弃全部自我批评。

## 激情、意志与行动

意志是我们意识到身体或心灵行动由命令产生时的内部印象，不是无原因实体能力。自由争论受词义混乱影响：必然性在人类行动中同样是动机、性格与行为的恒常结合，以及观察者由一者推到另一者的推论。

休谟把自由（*liberty*）界定为按意志行动或不行动的能力：若选择留下便可留下，选择离开便可离开。它与因果必然相容；若行动完全无原因、随机，就不能归属于性格，也难承担责任。

外在强迫取消这种自发自由。批评者会问，欲望若由操控或压迫形成，仅无外障是否足够；休谟相容论没有完整处理现代操控案例。

## 理性与激情

《人性论》说理性是且只应是激情的奴隶，意在否认理性单独产生最终目的。理性发现观念关系和因果事实，能告诉我手段、对象是否存在，却不在没有欲求时推动行动。

激情也不因与理性相反而虚假；当它依错误事实或选择无效手段时，相关信念可被纠正。理性对行动有重要工具和校正作用，但最终赞同、厌恶与目的来自情感。
