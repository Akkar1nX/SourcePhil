# 休谟总览

大卫·休谟（David Hume, 1711—1776）把「人的科学」置于哲学中心：先考察知觉、想象、习惯和情感如何实际工作，再限定形而上学可以提出什么。其怀疑论不是让人停止相信因果、外物和自我，而是说明这些不可避免信念并非由传统理性证明产生。

《人性论》（1739—1740）分知性、情感和道德三卷，初版反响不佳；《人类理智研究》（1748）重写认识论，《道德原则研究》（1751）重写道德论。《自然宗教对话录》因宗教风险在其死后 1779 年出版。后两部《研究》不是可忽略的通俗摘要，休谟本人称《道德原则研究》最完善。

## 哲学史位置

休谟继承洛克和贝克莱的知觉材料，却把经验来源原则用于因果力、自我实体和上帝观念；继承霍布斯对欲望的重视，却以同情和公共观点反对狭窄自利论。康德所说休谟使其摆脱「独断论迷梦」，主要关联因果必然性不能由经验重复给出；苏格兰常识哲学则认为休谟结论暴露观念理论和过度理由要求的问题。
