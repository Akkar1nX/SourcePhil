# 苏格兰常识哲学总览

苏格兰常识哲学（Scottish Common Sense philosophy）兴起于十八世纪中后期，主要人物包括托马斯·里德、詹姆斯·贝蒂、詹姆斯·奥斯瓦尔德、乔治·坎贝尔和杜格尔德·斯图尔特。它不是诉诸多数票或未经反思的偏见，而是主张推理、知觉、记忆和交往预设若干不可再由推论证明的第一原则；若哲学体系否定这些原则，应检查体系前提。

## 历史与制度背景

十八世纪苏格兰大学在格拉斯哥、爱丁堡、阿伯丁和圣安德鲁斯形成活跃课程，教会、法学、医学、修辞、道德哲学与自然科学相互联系。社团、评论刊物和英语写作扩大公共讨论。这一「苏格兰启蒙运动」还包括休谟、亚当·斯密、弗格森、凯姆斯等，不等于常识学派。

里德与贝蒂等人在阿伯丁哲学会交流。里德 1764 年出版《依常识原则探究人类心灵》，同年接替亚当·斯密任格拉斯哥道德哲学教授；1785 年《论人的理智能力》、1788 年《论人的主动能力》系统展开认识论和行动论。

## 与休谟的关系

里德认为休谟不是推理粗疏，而是把笛卡尔以来「观念是心灵直接对象」的共同理论推到一致结论。若我们只接触内部观念，物体、自我和因果便难以恢复。回应不应在观念桥梁上增加新证明，而应承认知觉本身直接指向外物、记忆直接承诺过去、人格同一先于记忆证据。

这种批评把争论从「怎样证明外物」转向「知觉的原初结构是什么」。康德同样回应休谟，却从先验条件而非常识第一原则出发；两条路线不能合并。
