# 关键争议与真题线索

## 贝克莱是主观唯心主义者吗

「主观唯心主义」可表示物体依赖心灵，却容易误导为物体依赖我的个人意识。贝克莱承认多重有限精神和无限精神，感觉秩序不由个人意志控制；他的世界不是私人梦。

若用这一标签，应说明「主观」指可感对象不能脱离任何精神，而非只有我的经验存在。否则会把非物质主义误写成唯我论。

## 上帝知觉能否保存对象

持续知觉读法说，上帝永远感知全部对象，所以对象持续存在。意志—规律读法指出，上帝不像有限者那样被动接受感觉；对象在神圣心灵中可能是意志内容或对有限者经验的稳定安排。

前者直观地解决无人房间，后者更符合上帝纯主动性。贝克莱文本同时使用「被上帝认识」和自然观念由上帝印入有限心灵的语言，解释尚有分歧。

## 非物质主义真的更符合常识吗

贝克莱认为常识桌子就是所见所触，不是不可知微粒基质，删除物质反而取消怀疑间隔。里德后来同意观念理论会导致贝克莱结论，却主张常识直接知觉独立物体，而不是把物体变为观念。

争议可表述为：直接性要求对象与知觉内容同一，还是知觉行为直接指向心外对象。两种直接实在论使用不同「直接」。

## 2021 真题：「存在即被感知」

答题先限定该命题只针对可感对象；再说明观念是被动内容，精神以感知和意志存在。论证可写：可感性质只作为观念被给予；第一性质不能脱离第二性质；观念只能像观念；未知物质基质既不可构想也无因果工作。因此物体是按规律结合的观念集合，其存在依赖被精神感知。

随后补上上帝：感觉观念不由有限意志产生，由无限精神按自然法给予，因此对象具有持续、秩序和公共性。最后可评价主论证是否混淆构想行为与内容，以及非物质主义是否真正保留常识外物。

只写「没人看桌子，桌子就不存在」是错误答案；贝克莱明确以神圣知觉或神圣观念秩序反对这种结论。

## 后续线索

休谟将保留观念／印象材料和对抽象、因果力的怀疑，却认为我们没有印象足以证明精神实体或神圣原因。里德则从贝克莱的论证学习到：若接受洛克式观念作为知觉直接对象，很难稳定保留物质，因此应拒绝「观念中介」而非外部世界。康德认为贝克莱把经验对象依赖认识条件误解为对象只是一组私人表象，自己的先验观念论需同时坚持经验实在论。
