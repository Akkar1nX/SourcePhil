# 贝克莱总览

乔治·贝克莱（George Berkeley, 1685—1753）接受心灵直接把握观念，却拒绝在观念背后再设不可感知的物质基质。可感物的存在就是被感知（*esse est percipi*），精神的存在则是感知和行动（可概括为 *esse est percipere*）。所以完整表述不是「一切存在都被感知」，而是区分被动观念与主动精神。

贝克莱生于爱尔兰，在都柏林三一学院学习，后任克洛因主教。《视觉新论》（1709）讨论距离知觉，《人类知识原理》（1710）提出非物质主义，《海拉斯和斐洛诺斯的三篇对话》（1713）回应怀疑主义，《阿尔西弗龙》（1732）批评自由思想家，《西里斯》（1744）则把焦油水、自然哲学和柏拉图主义联在一起。晚期思想不能只由 1710 年体系替代。

## 哲学史位置

贝克莱利用洛克的第二性质论和未知基质困难，主张第一性质也只能作为观念存在。他认为删除物质不是走向怀疑，而是取消夹在心灵与日常对象之间的不可知替身。休谟继承知觉材料和抽象批评，却不接受上帝及精神实体的同样保证；里德则认为贝克莱的结论暴露「观念作为知觉对象」理论本身有错。
